# financial-analysis-chatbot
📊 Financial Analysis Chatbot (BCG GenAI Simulation)

An AI-powered financial chatbot built as part of the BCG GenAI Job Simulation (Forage). The project uses Python, Flask, and Groq LLMs to analyze financial data and provide intelligent, conversational insights.

**🚀 Features**
AI chatbot powered by Groq LLM (LLaMA models)
Processes financial datasets from 10-K and 10-Q reports
Provides natural language financial insights
Maintains conversation history for context-aware responses
Rule-based + AI hybrid response system
Web-based interface using Flask
**🛠️ Tech Stack**
Python
Flask
Pandas
Groq API (LLM)
HTML/CSS (Frontend)
**📊 How it Works**
Loads financial dataset using Pandas
User enters financial queries via web interface
Chatbot processes input + dataset context
Groq LLM generates intelligent responses
Conversation history improves contextual understanding
**🎯 Use Case**

Designed to simulate real-world financial consulting AI tools, helping users analyze company financial data in a simple conversational format.

**🏆 Learning Outcome**
Python backend development
API integration (Groq LLM)
Financial data processing
Building AI-powered chat applications
Flask web application development
